Sistema de Autenticación en Python 🐍
* Este es un proyecto de lógica de programación que simula un sistema bancario/seguridad. 
* Me enfoqué en el manejo de errores y la validación de datos de usuario.

🚀 Características
* Validación de Usuario: Solo permite caracteres alfabéticos (evita números en el nombre).
* Seguridad: Sistema de 3 intentos para ingresar la contraseña.
* Manejo de Excepciones: Uso de bloques try-except para evitar cierres del programa por entradas inválidas.

🛠️ Tecnologías
* Lenguaje: Python 3.12
* IDE: Thonny

📌 Próximas Mejoras
